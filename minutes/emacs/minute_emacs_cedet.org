CEDET （ Collection of Emacs Development Environment Tools ） 是一个工具集，目标是在 Emacs 里创建一个高级开发环境。
它包含以下几个主要的组件：

 - EDE
 - Semantic
 - SRecode
 - Cogre
 - Speedbar
 - EIEIO
